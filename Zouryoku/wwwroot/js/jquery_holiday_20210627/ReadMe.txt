■jQuery UI datepicker 用祝祭日ツールチップ表示プラグイン
　jquery.holiday 20210627 α版
  
○機能
　日付選択ダイアログ datepicker にて日本の祝祭日名のツールチップを表示します。

○ファイル一覧
  js/jquery.holiday.js	祝祭日表示スクリプトJavaScriptJavaScript
  js/jquery.holiday.css	祝祭日のスタイルシート
  js/jquery.holiday_ja-JP.js	祝祭日のカスタマイズ例（日本語版）
  js/jquery.holiday_en-JP.js	祝祭日のカスタマイズ例（日本の祝祭日の英語版）
  sample.html	サンプルHTML（祝祭日機能の準備）
  sample.js		サンプルJS（祝祭日機能の呼び出し）
  js/jquery/配下	jquery, jquery-ui, datepicker など
